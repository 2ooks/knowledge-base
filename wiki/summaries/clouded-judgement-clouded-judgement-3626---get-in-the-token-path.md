# Clouded Judgement 3.6.26 - Get in the Token Path

**Source URL**: https://cloudedjudgement.substack.com/p/clouded-judgement-3626-get-in-the
**Date**: March 6, 2026
**Publication**: Clouded Judgement (Jamin Ball / Altimeter)

## Summary

Ball draws a direct parallel between cloud-era infrastructure winners and the emerging AI infrastructure landscape. The biggest cloud companies (Snowflake, Datadog, Cloudflare) succeeded by monetizing the core consumption primitive -- compute. In the AI era, that primitive is tokens. Companies whose revenue is structurally coupled to token generation and consumption (model labs, coding agents, inference providers) are the ones scaling fastest. Docker is the cautionary tale: it enabled billions in compute spend but never captured any of it. The advice: get in the token path, but build differentiation on top before per-unit economics compress.

## Key Claims

- The biggest cloud infrastructure winners monetized the core platform primitive (compute); AI's equivalent is tokens.
- Cursor reportedly hit $2B ARR with a business model structurally coupled to token consumption.
- Docker enabled billions in compute spend but failed to monetize the primitive -- Kubernetes and hyperscaler managed services captured the value instead.
- Limelight Networks had record traffic during 2020-2021 streaming boom but declining revenue -- being in the path without differentiation leads to commodity compression.
- Cloudflare vs. Limelight shows that being in the path is necessary but not sufficient; you must build switching costs on top.
- Inference costs are dropping rapidly, compressing per-unit economics -- the window to establish position in the token path is now.
- Companies with consumption pricing tied to platform-native units grew structurally with the ecosystem; seat-based models did not.
