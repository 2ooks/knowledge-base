# AI Engineering Goes Mainstream

**Source URL**: https://www.latent.space/p/aiewf-2025-keynotes
**Date**: June 13, 2025
**Publication**: Latent Space (swyx)

## Summary

Recap of the AI Engineer World's Fair 2025 (3,098 attendees, 306 speakers, 30+ workshops), which served as a milestone for AI engineering becoming a recognized discipline. Key launches included Gemini 2.5 Pro (onstage), Anthropic MCP Request for Startups, Claude Code Plan Mode, Dagger's container-use for agent sandboxing, and OpenAI Agents TypeScript SDK. Microsoft's Asha Sharma keynoted three strategic shifts: pair-to-peer programming (Copilot with autonomous MLE), software-to-agent factory (AI Foundry as an agent platform), and cloud-to-local AI (Foundry Local for on-device models). The conference signals that the agent infrastructure stack is crystallizing into distinct layers with clear partnership opportunities.

## Key Claims

- AIEWF 2025 drew 3,098 attendees (20% C-level, 17% management, 22% principal/staff eng) across 30 workshops and 10 tracks over 2 days.
- Microsoft's three-part thesis: pair-to-peer programming (Copilot assigned tasks autonomously), software-to-agent factory (AI Foundry with model routing, security, agentic RAG, MCP, observability), cloud-to-local (Foundry Local).
- Gemini 2.5 Pro launched onstage with roadmap for Gemini Code, Deep Research API, and Computer Use API.
- Anthropic issued an MCP Request for Startups and launched Claude Code Plan Mode.
- Dagger's container-use open-sourced containerized environments for agents -- a critical sandboxing primitive.
- Sarah Guo (Conviction): companies going from 0 to $100M run rate faster than any prior technology revolution.
- Simon Willison: MCP is "the most powerful technique in AI Engineering right now."
- OpenRouter framed as "inference-native middleware" complementing MCP -- input and output transforms for model routing.
- Wirth's Law reappears: distillation/inference/hardware gains are consumed by bigger models, reasoning chains, and agent loops.
