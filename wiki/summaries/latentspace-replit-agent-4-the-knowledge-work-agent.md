# Replit Agent 4: The Knowledge Work Agent

**Source URL**: https://www.latent.space/p/ainews-replit-agent-4-the-knowledge
**Date**: March 12, 2026
**Publication**: Latent Space

## Summary

Replit's tripling to a $9B valuation in six months reflects a broader 2026 trend: coding agent companies are expanding into general knowledge work (slides, videos, sites, canvas) now that software engineering is "approximately solved." The piece catalogs this pivot across Replit, Pi/OpenClaw, Claude Code/Cowork, and Notion Custom Agents, alongside a dense roundup covering NVIDIA Nemotron 3 Super (120B/12B-active open model), Anthropic's recursive self-improvement signals (70-90% of code written by Claude internally, 427x faster than human overseers), and a Claude Code auth outage that exposed the fragility of developer dependency on frontier model services.

## Key Claims

- Replit tripled in valuation to $9B in six months as it pivoted from coding platform to integrated knowledge work suite
- NVIDIA Nemotron 3 Super: 120B parameter / ~12B active, 1M context, hybrid Mamba-Transformer architecture, up to 2.2x faster inference than GPT-OSS-120B in FP4
- Anthropic reportedly has 70-90% of code for developing future models written by Claude, with release cadence compressed from months to weeks
- Claude is reportedly 427x faster than human overseers at some internal Anthropic tasks
- Karpathy argues the "death of IDE" framing is wrong -- instead "we need a bigger IDE" where the unit of work is an agent, not a file
- Claude Code auth outage described as potential "intelligence brownout" risk for engineering organizations
- PostTrainBench v1.0 released to benchmark whether frontier agents can post-train language models
- EvoSkill improved Claude Code + Opus 4.5 from 60.6% to 67.9% exact match on OfficeQA
