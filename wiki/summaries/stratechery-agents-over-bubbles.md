# Agents Over Bubbles

**Source URL**: https://stratechery.com/2026/agents-over-bubbles/
**Date**: March 16, 2026
**Publication**: Stratechery

## Summary

Thompson argues we are not in an AI bubble, presenting a three-paradigm framework: ChatGPT (readable AI), o1 (reliable AI), and Opus 4.5/Claude Code (autonomous agents). The agentic paradigm is the inflection point that justifies hyperscaler capex because agents multiply compute demand without requiring mass consumer adoption -- a small number of people with agency can drive massive compute utilization. Crucially, agent differentiation comes from model-harness integration (not models alone), which means Anthropic and OpenAI are integrated value chain leaders, not commodity providers -- directly undermining the "models are commodities" thesis and validating their sky-high valuations.

## Key Claims

- Three LLM paradigms: ChatGPT (Nov 2022), o1 (Sep 2024), Opus 4.5 + Claude Code (late 2025) -- each dramatically increasing compute demand.
- Agent workloads require CPUs in addition to GPUs for tool invocations, data lookups, and orchestration.
- Agent differentiation comes from model-harness integration, not the model alone -- Anthropic's Claude Code harness changes made Opus 4.5 suddenly dramatically more useful.
- Microsoft abandoned model-agnosticism for Copilot Cowork, which only works with Claude, admitting agents need integrated model+harness.
- The MacBook Neo at $599 was described by Asus CFO as "a shock to the entire market."
- Enterprise economic imperatives will drive layoffs, with companies rebuilding scale using agents instead of humans.
- Anthropic and OpenAI are the integration points in the AI value chain, making their valuations more defensible than commodity model providers.
