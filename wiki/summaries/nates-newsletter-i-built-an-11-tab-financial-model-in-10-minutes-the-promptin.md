# I Built an 11-Tab Financial Model in 10 Minutes: The Prompting Guide That Makes It Repeatable

**Source URL**: https://natesnewsletter.substack.com/p/anthropic-just-put-claude-inside
**Date**: January 27, 2026
**Publication**: Nate's Newsletter (Substack)

## Summary

Anthropic launched Claude as a native Excel sidebar add-in with full workbook structural awareness, cell-level citations, and transparent change tracking -- compressing weeks of financial modeling work into minutes. The deeper strategic argument is that base model competition is hitting diminishing returns and the real battleground has shifted to workflow integration backed by proprietary data partnerships. Anthropic assembled licensed data feeds from LSEG, Moody's, S&P Capital IQ, FactSet, and others via MCP, creating a data moat that Microsoft cannot easily replicate. Meanwhile, Microsoft is simultaneously hosting Claude, competing with it via Copilot, and collecting $30B from Anthropic for Azure infrastructure.

## Key Claims

- An 11-tab financial model (sensitivity analysis, opportunity cost, risk quantification, market data, cash flow projections) was built in 10 minutes using Claude in Excel.
- Norway's sovereign wealth fund reported saving 213,000 hours using Claude; production deployments at Citi, Deutsche Bank, and Bridgewater.
- Anthropic opened Claude in Excel to anyone with a $20/month Pro subscription on January 24, 2026.
- Anthropic assembled licensed data partnerships via MCP with LSEG, Moody's, S&P Capital IQ, FactSet, Morningstar, PitchBook, and Aiera.
- Microsoft and Anthropic signed a $30B partnership; Claude is available in Copilot Studio and powers the M365 Researcher agent.
- Copilot for Excel requires OneDrive/AutoSave (cloud-first); Claude in Excel works with local files -- a product gap created by Microsoft's own strategy.
- METR benchmark shows frontier model task endurance approaching 3 hours, up from 1 hour a year ago -- the threshold that makes workflow integration viable.
- GPU compute scaled 106x since 2016 (P100 to B200) but memory bandwidth only 11x -- the "memory wall" sustains NVIDIA's pricing power.
- The model quality race is approaching diminishing returns; competition has shifted to workflow integration and data partnerships.
