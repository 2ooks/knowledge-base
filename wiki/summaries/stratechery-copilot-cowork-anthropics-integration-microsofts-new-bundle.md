# Copilot Cowork, Anthropic's Integration, Microsoft's New Bundle

**Source URL**: https://stratechery.com/2026/copilot-cowork-anthropics-integration-microsofts-new-bundle/
**Date**: March 10, 2026
**Publication**: Stratechery

## Summary

Microsoft launched Copilot Cowork -- built on Anthropic's Claude Cowork technology -- as an enterprise agent that runs in the cloud within M365's security boundary, paired with a new E7 bundle at $99/seat/month (65% above E5). This is a landmark strategic admission: Microsoft abandoned model-agnosticism for its most important new product because agents require integrated model+harness, and Anthropic uniquely provides that integration. Thompson frames this as Microsoft executing "commoditize your complements" at every layer except the one that matters most -- the agentic harness -- where Anthropic holds the integration point and likely commands healthy margins.

## Key Claims

- Copilot Cowork is based on Anthropic's Claude Cowork technology, running only on Claude models (not model-agnostic).
- Microsoft's new E7 bundle costs $99/seat/month, a 65% increase over E5.
- Copilot Cowork runs exclusively in the cloud within M365 security boundaries, unlike Claude Cowork which runs locally.
- Microsoft has added Claude Sonnet models to M365 Copilot alongside OpenAI's GPT models.
- Over 1,500 customers have used both Anthropic and OpenAI models on Foundry.
- Work IQ grounds agent actions in organizational data -- people, roles, artifacts, communications, history -- within the security boundary.
- Active Directory's identity ownership is Microsoft's starting point for the agent platform, but per-seat value shrinks if headcount declines.
- Microsoft expects to expand E7 with governance, security, and additional AI features to justify the price increase over time.
