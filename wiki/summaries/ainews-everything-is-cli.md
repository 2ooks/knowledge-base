# AINews: Everything is CLI
- **Source:** https://www.latent.space/p/ainews-everything-is-cli
- **Date:** March 26, 2026
- **Author:** Latent.Space (swyx)

## Summary
Stripe launched Projects.dev, a CLI for agents to instantly provision backend services, on the same day as CLIs from Ramp, Sendblue, Kapso, ElevenLabs, and Visa. The piece argues CLIs are becoming the default interface for agent-native infrastructure, potentially more practical than MCPs for many use cases. The broader roundup covers Gemini 3.1 Flash Live, Mistral Voxtral TTS, Cline Kanban for multi-agent orchestration, and the emerging "harness engineering" thesis that middleware/memory/tooling around base models is the real product.

## Key Claims
- CLIs are emerging as the default agent-to-service interface, displacing or complementing MCP
- "Harness engineering" — the agent harness (middleware, memory, task orchestration) — is increasingly the real product, not the base model
- Cline Kanban solves the two practical bottlenecks of coding agents: inference-bound waiting and merge-conflict parallelism
- Cursor ships improved checkpoints every 5 hours via "real-time RL" — continual learning in production
- ProRL Agent (NVIDIA): decoupling RL rollout nearly doubled SWE-Bench Verified scores
