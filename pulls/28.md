# Add mobile-first static wiki UI with GitHub Pages deployment

This pull request introduces a mobile-first static wiki UI, designed to be deployed using GitHub Pages. The following features are included:

- Responsive design for mobile devices
- Easy navigation through wiki contents
- Deployment guide for GitHub Pages

## Implementation Details
- The UI is built using HTML, CSS, and JavaScript
- Uses a modern framework for seamless deployment

## Deployment Instructions
1. Clone the repository
2. Navigate to the `/docs` folder
3. Run `npm install` to install dependencies
4. Use `npm run deploy` to push to GitHub Pages.

This PR aims to streamline the user experience on mobile devices, making the information more accessible to users on the go.
