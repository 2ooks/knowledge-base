# Project Glasswing: Securing critical software for the AI era

**Source:** https://www.anthropic.com/glasswing
**Date:** April 7, 2026
**Retrieval:** Secondary sources only (anthropic.com blocked)

## Secondary Source Summary

Anthropic's Project Glasswing is a consortium-based cybersecurity initiative that provides major tech companies with access to Claude Mythos Preview, an unreleased frontier AI model specialized for vulnerability discovery. The project aims to shift the advantage to defenders by enabling automated, AI-powered security scanning of critical software infrastructure.

### Key Details from Secondary Sources

- **Consortium members:** AWS, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, Microsoft, NVIDIA, Palo Alto Networks, Linux Foundation, and ~40 other organizations
- **Model capabilities:** Claude Mythos Preview can detect vulnerabilities more effectively than all but the most skilled human security experts
- **Early results:** Already found thousands of high-severity vulnerabilities, including a 27-year-old bug in OpenBSD and a 16-year-old FFmpeg vulnerability
- **Access model:** Restricted to consortium members and critical software maintainers—NOT released publicly
- **Rationale for restricted access:** Dual-use concerns—the model could enable malicious actors to find and exploit vulnerabilities at scale
- **Financial commitment:** $100 million in Mythos Preview usage credits + $4 million in direct donations to open-source security organizations
- **Name origin:** "Glasswing" butterfly with transparent wings—metaphor for making invisible vulnerabilities visible

## Strategic Implications

- First major frontier model with explicit dual-use restriction (not even API access)
- Cross-competitor consortium including Microsoft, Google, Apple, AWS working together
- Shifts vulnerability economics from human-centric bug bounties to AI-scale automated discovery
- "Window of exposure" becomes critical metric as AI accelerates both discovery and patching
- Model capabilities reveal Mythos as significantly more capable than Opus 4.6 (previously Anthropic's public frontier)
